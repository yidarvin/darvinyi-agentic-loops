4. **`artifacts/ch21-stage-three-production-grade/stage_three_agent.py`: the MCP server can rewrite the tool-definition lock that is supposed to detect its drift.** The lock lives under workspace memory (lines 406-414), while the Seatbelt profile grants the server write access to the whole workspace (lines 191-194) and gives it the workspace path.
